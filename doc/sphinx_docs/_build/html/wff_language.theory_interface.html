class wff_language.theory_interface.Language(parser)[source]¶

Bases: object

A language over an alphabet of symbols. Currently defined by providing a parse-function for the language. Note that the parse function represents the grammar of the language, and either returns an abstract syntax tree (AST) or reports failure when passed an expression. The parsing defines which expressions are well-formed. The AST assigns a grammar-part label to each subtree. The parse function may itself include sub-languages in its definition. (Later, with automatic parser-generation, the grammar alone could be used in the specification.)
