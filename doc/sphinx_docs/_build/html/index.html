Typped: A typed Pratt parser employing dispatching¶

This package provides a general purpose Python framework for creating Pratt parsers to parse possibly typed target languages. It provides default parser functions for many standard constructs (such as infix operators and ordinary functions) and can be extended to allow more complicated or specific constructs to be parsed.

Installation¶

The easiest way to install Typped is to use pip:

pip install typped
